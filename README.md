# tensorflow-test

Test tensorflow after installing CUDA GPU support

## Requirements

- [Quarto](https://quarto.org/) — `[scoop](https://scoop.sh/) install quarto`
- [Pyenv]()
- [Pipenv]()
- Python 3.10.5 — `pyenv install 3.10.5`

## How to run

```bash
pyenv exec pipenv install
pyenv exec pipenv shell
```
